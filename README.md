# Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production

This repository contains data and partial analysis code associated with the paper entitled “Sewage Sludge as Substrate in *Schinus terebinthifolia* Raddi Seedling Commercial Production,” published in *Scientific Reports* in October 2022 (Alonso et al., 2022; https://doi.org/10.1038/s41598-022-21314-0).

## Analysis report

The rendered analysis report, including R code, outputs, and figures, is available here:

[Open the analysis report](https://jmalonso55.github.io/substrate_fertilier_schinus/)

## Repository structure

- `docs/index.html`: rendered HTML report.
- `analysis.Rmd`: source R Markdown file used to generate the report.
- `stat_aroeira.csv`: dataset used in the analyses.
- `README.md`: repository description.

## Citation

Alonso, J. M.; et al. Sewage sludge as substrate in *Schinus terebinthifolia* Raddi seedlings commercial production. *Scientific Reports*, 2022. https://doi.org/10.1038/s41598-022-21314-0
